exercise1 /foo/bar restful webservice

to run this project\
    - must have java (preferably 8) and maven installed
    - run mvn clean package\
    - launch app from command line\
    - run "java -jar target/exercise1.jar" command line\
    - once you see app started and listening on port 8080\
    - from postman do a GET to http://localhost:8080/foo/bar\
    - from postman do a GET to http://localhost:8080/foo/bar/json
